# Terminal Commands
## Linux Generals
### 5. ls (Lidt directories)
- ls -l: prints files with timestamp
### 5. touch (Create files)
- touch file.txt: Creates a file named file.txt
### 13. rm (Remove)
- rm file: Remove a file.
- rm -f file: Force the removal of file whether they are protected or not.
- rm -i file: gives a promp on wether you actually want to remove the files or directory.
- rm -r diretory: Removes a directory.
- rmdir directory: Also removes a directory.

- g: Go to URL
