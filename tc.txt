### 13. rm (Remove)
- rm file: Remove a file.
- rm -f file: Force the removal of file whether they are protected or not.
- rm -i file: gives a promp on wether you actually want to remove the files or directory.
- rm -r diretory: Removes a directory.
- rmdir directory: Also removes a directory.

## Git
### Basic push workflow
- git remote -v: Print the remote origin URL
- git add file.txt: Stages "file.txt"
- git add .: Stages everything but the thing in the .gitignore file
- git commit -m "Your commit message": 
	路 Captures the current state of the staged files
	路 Links this snapshot to the previous commit, forming a 	 chain (the project's history )
	路 Saves a reference (a pointer) to this commit in the 
	  repository
	路 The -m flag allows you to include a message directly
	  in the command
